# Image Recognition Project

## Description
This project uses Google Teachable Machine to train an image recognition model and Google Colab to test the exported TensorFlow Keras model using Python.

## Classes
- Eagle (نسر)
- Falcon (صقر)
- Golden Eagle (عقاب)

## Model Evaluation
A different image that was not included in the training dataset was used to evaluate the model. The model successfully predicted the correct class.

## Tools Used
- Google Teachable Machine
- Google Colab
- Python
- TensorFlow / Keras

## Files Included
- predict.py – Python script used to load the model and predict the image class.
- keras_model.h5 – Trained TensorFlow Keras model.
- labels.txt – Class labels generated by Teachable Machine.
- output_screenshot.png – Screenshot showing the prediction result.
